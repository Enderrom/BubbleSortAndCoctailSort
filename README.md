# Сравнение алгоритмов сортировки: Bubble Sort vs Cocktail Shaker Sort

Проект для сравнения производительности двух алгоритмов сортировки на различных размерах массивов со случайными числами.

## О проекте

Этот проект реализует и сравнивает время выполнения двух алгоритмов сортировки:
- **Bubble Sort** (сортировка пузырьком)
- **Cocktail Shaker Sort** (шейкерная сортировка)

Цель проекта - наглядно продемонстрировать разницу в производительности этих алгоритмов на массивах разного размера (от 10 до 10000 элементов).

## Особенности

- Реализация двух алгоритмов сортировки на C++
- Тестирование на массивах 10 различных размеров
- Использование высокоточного таймера (`chrono`)
- 10 000 итераций для каждого размера массива для повышения точности
- Чтение случайных чисел из файла `random_numbers.txt`
- Вывод среднего времени выполнения в секундах

## Компиляция и запуск

### Предварительные требования

- Компилятор C++ с поддрежкой C++14
- Файл "random_numbers.txt" с тестовыми данными

### Шаги для запуска

- Перед компиляцией стоит изменить значение iterators на 1000 (в репозиторий загружен уменьшенный файл, где массивов для каждого размера данных уменьшен с 10000 до 1000)
- Распаковать "rundom_numbers_test.zip" и поместить его в папку с "Source.cpp"
- Открыть терминал в папке с "Source.cpp" и "ranadom_numbers_test.txt"
- Для компиляции 'g++ Source.cpp -o Source.exe'
- Для запуска '.\Source.exe'
